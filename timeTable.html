Timetable System (Web/ Mobile Application)

  • Web Application developed using Java Spring Boot, Thymeleaf. Database functionality carried out by MySQL
  • CRUD functions, Login, Register, Accessing Timetable,Admin Panel, Reserving certain classes
  • Java Spring Boot was used to develop the backend and Thymeleaf and Bootstrap to develop the frontend
  • A android application was made for the same functions using Android Sutdio, with the use of JSON APi's the web application was linked with the mobile application to perform the same tasks.
